# Movie Watchlist (JavaScript)

A simple web application where users can search for movies and save their favorites to a watchlist.

---

## Project Idea

This project allows users to search for movies and manage a personal watchlist. Movie data will be fetched from an external API.

---

## Purpose

The purpose of this project is to:

* Learn JavaScript fundamentals
* Understand how to use APIs with fetch()
* Build an interactive and user-friendly web application
* Practice array methods like map(), filter(), and sort()

---

## API Used

This project uses the **OMDb API (http://www.omdbapi.com/?i=tt3896198&apikey=fb6234a5)**.

* Provides movie title, poster, and year
* Supports search functionality
* Free and easy to use

---

## Planned Features

### Core Features

* Search movies
* Display search results
* Add movies to watchlist
* Remove movies from watchlist
* Show message when no results are found

### Future Enhancements

* Filter movies
* Sort movies
* Random movie suggestion
* Dark mode UI
